An Excel dashboard analyzing Electric Vehicle (EV) sales data across India from 2014 to 2023.

# Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Data Cleaning (removed zero-sales rows, standardized vehicle types)

# Dataset
- 95,985 raw records cleaned down to 13,812 active sales rows
- Covers Indian states and UTs
- Years: 2014 – 2023
- Vehicle types: 2-Wheelers, 3-Wheelers, 4-Wheelers, Bus

# Key Insights
- Total EV sales crossed **35.9 lakh units** across the dataset
- Sales grew **638x** from 2,392 units in 2014 to 15,25,179 in 2023
- **2-Wheelers** dominate with 50%+ of total category sales
- **Uttar Pradesh** is the top state for EV adoption
- Massive growth spike visible post-2021 — driven by government FAME-II policy

# Dashboard Features
- Category-wise sales breakdown (pie chart)
- Year-wise sales trend (line chart)
- Month-wise performance (bar chart)
- State-wise comparison (bar chart)
- Vehicle-type area chart (stacked)
- Interactive slicers for Year, Month, and Vehicle Category
